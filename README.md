# Optimización de Rutas Logísticas Mediante el Algoritmo de Held-Karp Usando Programación Dinámica

Este repositorio contiene la implementación y comparación analítica de dos enfoques algorítmicos para resolver el 
**Problema del Viajante de Comercio (Traveling Salesman Problem - TSP)**.
